RollCall帮助文档 ​

程序概述 ​

RollCall 是一个基于 WPF 的点名程序，旨在通过随机抽取的方式实现高效、公平的点名操作。它支持从本地文件（如 TXT、Word、Excel）导入名字列表，并确保在一个轮次中每个名字只被抽取一次。程序还提供了系统托盘图标、右键菜单以及简单的图形界面，方便用户操作。

功能特点 ​

  • 多文件支持：支持从 TXT、Word 和 Excel 文件导入名字列表。

  • 随机点名：确保每个名字在一个轮次中只出现一次。

  • 系统托盘图标：支持最小化到系统托盘，并通过右键菜单操作。

  • 更新提示：检测 GitHub 上的最新版本，并提示用户更新。

  • 图形界面：提供简单的用户界面，方便用户操作。
